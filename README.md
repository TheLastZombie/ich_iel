## [Running on your Machine](../../wiki/Run-ðŸ‡ºðŸ‡¸)

<details><summary>Manually</summary>
<p>

```
git clone https://github.com/TheLastZombie/ich_iel
cd ich_iel
npm install
```

Replace the environment variables in `ich_iel.js`.

```
npm start
```

</p>
</details>

<details><summary>dotenv</summary>
<p>

```
git clone https://github.com/TheLastZombie/ich_iel
cd ich_iel
npm install
```

Replace the environment variables in `.env`.

```
node -r dotenv/config ich_iel.js
```

</p>
</details>

<details><summary>Heroku</summary>
<p>

```
git clone https://github.com/TheLastZombie/ich_iel
cd ich_iel
npm install
```

Replace the environment variables in `.env`.

```
heroku local
```

</p>
</details>
